## üìê MATHEMATICS BEHIND THE VISUALS

### **2D Fractals (Mandelbrot Set)**
```
Z(n+1) = Z(n)¬≤ + C

Where:
- C is a complex number (x + yi)
- Z starts at 0
- Iterate until |Z| > 2 or max iterations
- Color based on iteration count
- Zoom based on energy level
- Center based on stress + mood
```

### **3D Particle Distribution (Fibonacci Sphere)**
```
For particle i of N total:
phi = arccos(1 - 2*(i + 0.5)/N)
theta = GOLDEN_ANGLE * i

x = radius * sin(phi) * cos(theta)
y = radius * sin(phi) * sin(theta)
z = radius * cos(theta)

Where:
- GOLDEN_ANGLE = 137.5077640500378¬∞
- radius = 3 + noise*2*energy + (1-stress)*2
- noise = (sin(theta*8)*cos(phi*8) + 1)/2
```

### **Fibonacci Spiral (3D)**
```
For point i:
angle = i * GOLDEN_ANGLE
radius = sqrt(i) * PHI * 0.5

x = radius * cos(angle)
y = radius * sin(angle)
z = i * 0.05 - 2.5

Where:
- PHI = 1.618033988749895 (Golden Ratio)
- Creates spiral that converges to PHI
```

### **Color Mapping (Autism-Safe)**
```
RGB values for particle i:
R = 0.3 + mood * 0.2       # Blues (calm)
G = 0.5 + energy * 0.3     # Greens (nature)
B = 0.6 + (1-stress) * 0.3 # Deep blues (peaceful)

Result: Always calm blues/greens, never red/orange
```

## ğŸ› TROUBLESHOOTING

### "Ollama not available"
**Problem:** Pet assistant says "thinking..." but no response
**Solution:** 
1. Install Ollama: https://ollama.ai
2. Start Ollama: `ollama serve`
3. Pull model: `ollama pull llama3.1`
4. Set OLLAMA_API_URL in environment

### "Whisper not available"
**Problem:** Voice transcription fails
**Solution:**
1. Install: `pip install openai-whisper`
2. Or use text input instead
3. Frontend will fallback to text automatically

### "No celebration sound"
**Problem:** Task completes but no audio
**Solution:**
1. Check browser allows audio
2. Click page first (browsers block autoplay)
3. Check /api/audio/celebration endpoint directly

### "Report is empty"
**Problem:** Progress report shows zeros
**Solution:**
1. Create some tasks first
2. Complete at least one task
3. Check database has data
4. Refresh and try again
